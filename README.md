# AutoScripts

实现各种功能的自动化脚本  
The scripts used to do things automatically

## AppleidSites
> 通过selenium实现appleid官网的操作的脚本，包括：登陆、修改密码、注销等  
> autoscripts using selenium for appleid official website actions, including: sign in, update password, sign out, etc.
